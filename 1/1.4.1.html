while 문장은 주어진 조건이 참일 때까지 코드의 한 부분을 반복적으로 실행한다. 우리는 while을 1에서 10까지의 수를 더하는 프로그램에 쓸 수 있다. 
#include <iostream>
int main()
{
    int sum = 0, val = 1;
    // keep executing the while as long as val is less than or equal to 10
    while (val <= 10){
        sum += val;   // assigns sum + val to sum
        ++val;        // add i to val
    }
    std::cout << "Sum of 1 to 10 inclusive is "
              << sum << std::endl;
    return 0;
}
이 프로그램을 컴파일하면, 다음이 출력된다. 
Sum of 1 to 10 inclusive is 55

이전과 마찬가지로, 우리는 iostream 헤더 포함 및 main 정의로 시작한다. main 안에서 우리는 int 변수 두 개를 정의한다. 합을 담을 sum과 1부터 10에 이르는 값을 표현할 val이 그것이다. 우리는 sum을 0으로 초기화하고 val을 1의 값으로 시작한다.

프로그램에서 새로운 부분은 바로 while 문장이다. while은 다음 형식을 취한다. 
while(condition)
        statement

while은 조건을 검사하고 조건이 거짓일 때까지 연결된 문장을 실행함으로써 실행된다. 조건(condition)이란 결과물이 참인지 거짓인지 결정하는 표현식이다. 조건이 참일 때, 문장은 실행된다. 문장 실행 이후, 조건은 다시 검사된다. 조건이 다시 참이라면, 문장은 다시 실행된다. while은 계속되며, 조건이 거짓일 때까지 조건을 검사하고 문장을 실행한다.

이 프로그램에서, while 문장은 다음과 같다.

// keep executing the while as long as val is less than or equal to 10
while(val <= 10){
    sum += val;   // assigns sum + val to sum
    ++val;        // add 1 to val
}
조건은 이하(<=) 연산자 연산자를 사용하여 val과 10의 현재 값을 비교한다. val이 10 이하인 이상, 조건은 참이다. 조건이 참이라면, while 몸체가 실행된다. 이 경우 몸체란 다음 두 문장의 블록이다. 
{
    sum += val;   // assigns sum + val to sum
    ++val;        // add 1 to val
}
블록이란 중괄호로 묶이는 0개 이상의 일련의 문장이다. 블록 또한 하나의 문장으로, 문장이 요구되는 곳이라면 어디든 쓰일 수 있다. 이 블록의 첫 번째 문장은 복합 대입 연산자(compound assignment operator, +=)를 사용한다. 이 연산자는 우측 피연산자를 자신의 좌측 피연산자에 더한 다음 그 결과를 좌측 피연산자에 저장한다. 본질적으로 다음처럼 덧셈을 수행하는 것과 다를 바가 없다. 
sum = sum + val;   // assign sum + val to sum

따라서 이 블록상의 첫 번째 문장은 sum의 현재 값에 val의 값을 더한다.

다음 문장은 다음과 같다. 

++val;    // add 1 to val;

이 문장은 전위 증감 연산자(prefix increment operator, ++)를 사용한다. 증감 연산자는 피연산자에 1을 더한다. ++val이라고 적는 일은 val = val + 1이라고 적는 것과 같다.

while 몸체 실행이 끝나면 루프는 다시 조건을 평가한다. (방금 증감된) val의 값이 여전히 10 이하라면, while 몸체가 다시 실행된다. 루프는 계속되고, val이 더는 10 이하는 아닐 때까지 조건을 검사하고 몸체를 실행한다.

val이 10을 초과할 때면, 프로그램은 while 루프 밖으로 떨어져 나간 다음 while에서 이어지는 문장을 이어서 실행한다. 이 경우, 그 문장은 바로 출력을 담당하는 것으로, 그다음에는 return이 이어져, main 프로그램이 끝난다.
