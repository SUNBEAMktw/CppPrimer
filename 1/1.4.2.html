while 루프에서는 val 변수를 이용해 루프 실행 횟수를 제어했다. 또 val의 값을 조건상에서 검사했고 while 몸체 내부에서 val을 증가했다.

이처럼 변수를 조건 내부에서 사용하고 그 변수를 몸체 내부에서 또 사용하는 패턴은 너무 자주 일어나기에, 언어는 두 번째 문장을 정의한다. 그것이 바로 for 문장으로, 이 패턴을 따르는 코드를 요약한다. 우리는 for 루프를 이용해 프로그램을 재작성하여 1에서 10까지의 숫자를 다음처럼 합칠 수 있다.

#include <iostream>
int main()
{
    int sum = 0;
    // sum values from 1 through 10 inclusive
    for (int val = 1; val <= 10; ++val)
        sum += val;   // equivalent to sum = sum + val
    std::cout << "Sum of 1 to 10 inclusive is "
              << sum << std::endl;
    return 0;
}

저번처럼 이번에도 sum을 정의한 다음 이를 0으로 초기화한다. 이 판본에서, 우리는 val을 for 문장 자체의 일부분으로 정의한다.

for (int val = 1; val <=10; ++val)
    sum += val;

각 for 문장은 두 부분으로 구성된다. 헤더와 몸체가 그것이다. 헤더는 몸체 실행 빈도를 제어한다. 헤더 자체는 세 부분으로 구성된다. 초기화 문장(init-statement), 조건(condition), 그리고 표현식(expression)이 그것이다. 이 경우, 초기화 문장은 다음과 같으며

int val = 1;

val이라는 이름의 int 객체를 정의하고 여기에 1이라는 초깃값을 부여한다. val 변수는 오직 for 내부에만 존재한다. 루프 종료 이후 val을 사용하는 것은 불가능하다. 초기화 문장은 for 시작과 함께 단 한 번 실행된다. 조건문은 다음과 같으며 

val <= 10
val의 현재 값을 10과 비교한다. 조건문은 루프가 돌 때다마다 검사된다. val이 10 이하인 이상, for 몸체가 실행된다. 표현식은 for 몸체 이후에 실행되며, 다음과 같다 
++val

전위 증감 연산자를 사용해, val 값에 1을 더한다. 표현식 실행 이후, for는 조건을 재검사한다. 새 val 값이 여전히 10 이하라면, for 루프 몸체는 다시 실행된다. 실행 이후, val은 다시 증감된다. 루프는 조건이 실패할 때까지 이어진다.

이 루프에서 for 몸체는 합산을 수행한다. 
sum += val;  // equivalent to sum = sum + val

복습하자면, 여기서 for는 전체적으로 이렇게 실행된다.

  1. val 실행 및 1로 초기화.
  2. val이 10 이하인지 검사. 성공시, for 몸체 실행. 실패시, 루프 탈출 및 for 몸체에 뒤따르는 첫 문장 실행.
  3. val 증감.
  4. 2단계의 검사 재실행. 조건이 참이면 남은 단계 진행.
