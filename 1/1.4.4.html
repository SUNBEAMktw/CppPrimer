여타 언어와 마찬가지로 C++는 조건적 실행을 지원하는 if 문장을 제공한다. if를 이용해 입력상에서 어떤 값이 얼마나 반복되었는지 확인하는 프로그램을 작성할 수 있다.

#include <iostream>
int main()
{
	// currVal is the number we’re counting; we’ll read new values into val
	int currVal = 0, val = 0;
// read first number and ensure that we have data to process
if (std::cin >> currVal){
int cnt = 1;	// store the count for the current value we’re processing
while (std::cin >> val){		// read the remaining numbers
if (val == currVal) 	// if the values are the same
	++cnt;		// add 1 to cnt
else {	// otherwise, print the count for the previous value
	std::cout << currVal << “ occurs “
		<< cnt << “ times” << std::endl;
	currVal = val;		// remember the new value
	cnt = 1;		// reset the counter
}
}	// while loop ends here
// remember to print the count for the last value in the file
std::cout << currVal << “ occurs “
	<< cnt << “ times” << std::endl;
	} // outermost if statement ends here
	return 0;
}

여기 아래 입력을 준다면

42 42 42 42 42 55 55 62 100 100 100

출력물은 다음처럼 나와야 한다.

42 occurs 5 times
55 occurs 2 times
62 occurs 1 times
100 occurs 3 times

대부분 이전 프로그램을 통해 익숙한 코드일 것이다. 우리는 우선 val과 currVal을 정의하는 것으로 시작한다. currVal은 우리가 세고 있는 숫자를 파악할 것이다. val은 입력에서 읽어 들이는 각각의 숫자를 취할 것이다. 여기서 새로운 것은 두 if 문장이다. 첫 번째 if는 다음과 같으며

if (std::cin >> currVal) {
	// …
}	// outermost if statement ends here

입력이 비지 않았다는 점을 보장한다. while과 마찬가지로 if는 조건을 평가한다. 첫 번째 if 안의 조건은 currVal로 값을 읽어 들인다. 읽는 데 성공했다면 조건은 참이며 조건에서 이어지는 중괄호로 시작하는 블록이 실행된다. 이 블록은 return 문장 직전에 있는 닫는 중괄호와 함께 끝난다.

셀 숫자가 있다는 것을 알았다면 우리는 cnt를 정의해 각각의 숫자가 나타난 횟수를 센다. 우리는 이전 단원에서 본 while 루프와 유사한 것을 사용해 표준 입력에서 숫자를 (반복적으로) 읽어 들인다.

while의 몸체는 두 번째 if 문장을 포함하는 하나의 블록이다.

if (val == currVal)	// if the values are the same
	++ cnt;		// add 1 to cnt
else {	// otherwise, print the count for the previous value
	std::cout << currVal << “ occurs “
		<< cnt << “ times” << std::endl;
	currVal = val;	// remember the new value
	cnt = 1; 	// reset the counter
}

이 if 속 조건은 (==)상등 연산자를 사용해 val이 currVal과 상등인지 검사한다. 상등이라면 조건에서 바로 이어지는 문장이 실행된다. 그 문장은 cnt를 증가시키며, currVal가 한 번 더 포착되었다는 것을 암시한다.

조건이 거짓이라면, 즉, val이 currVal이 상등이 아니라면, else에서 이어지는 문장을 실행한다. 그 문장은 출력문과 대입문 두 개로 구성되는 하나의 블록이다. 이 출력문은 방금 막 처리를 끝낸 값을 센 횟수를 출력한다. 대입문은 cnt를 1로 재설정하고 currVal을 val로 재설정한다. 여기서 val은 방금 우리가 읽은 숫자다.

주의

C++는 =를 대입에 쓰고 ==를 상등에 쓴다. 두 연산자 모두 조건 안에 나타날 수 있다. ==를 써야할 때 =를 쓰는 것은 흔한 실수다.

핵심 개념: C++ 프로그램의 들여쓰기와 형식화

넓게 말해 C++ 프로그램은 자유 형식이다. 즉 중괄호, 들여쓰기, 주석, 개행 등이 프로그램의 뜻에 큰 영향을 미치지 않는다. 가령 main 몸체가 시작한다는 것을 표기하는 중괄호는 main과 같은 라인에 있어도 상관 없다. 다음 라인의 시작이나, 어디든 상관 없다. 유일한 조건은 main의 매개변수 목록 이후의 첫 번째 비공백, 비주석 문자 이후여야 한다는 것이다.

다만 프로그램 가독성이 문제다. 우리는 물론 main 전체를 하나의 라인에 쓸 수도 있다. 하지만 그럴 경우 가독성이 떨어질 것이다.

C++ 프로그램 형식화에 관한 논쟁은 끊이지 않는다. 우리는 하나의 옳은 방법이 있다고 믿지 않는다. 다만 일관성은 좋은 것이다. 대다수 프로그래머는 프로그램의 부수적인 부분에서 들여쓴다. 우리가 main 내부 문장과 루프의 몸체에서 그랬던 것처럼 말이다. 우리는 함수의 범위를 정하는 중괄호에게 제 라인을 부여한다. 우리는 또한 연산자들이 이어지도록 복합 입출력 표현식을 들여쓰기한다. 프로그램이 복잡해질수록 다른 들여쓰기 관행 또한 명백해질 것이다.

유의할 점은 다른 형식화 또한 가능하다는 점이다. 형식화 스타일을 선택할 때는, 가독성과 이해도를 신경 써라. 스타일을 골랐으면, 일관성을 지켜라.
